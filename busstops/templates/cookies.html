Cookies

‘Cookies’ are small text files stored on your computer when you use some websites, including this one.

We use some cookies for the functionality of this website:

Name Type Purpose
vehicleMap local storage Last viewed location on the map
hideStops local storage Whether to show bus stops on the map
sessionid cookie Remember if you have logged in

We use Google Analytics to measure website usage, which sets these cookies:

Name Default expiration time Purpose
_ga 2 years Distinguish users
_ga_<container-id> 2 years Persist session state
